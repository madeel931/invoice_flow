# InvoiceFlow Pro - Offline-First Flutter Billing & CRM App

**InvoiceFlow Pro** is a premium, production-ready Flutter application designed for freelancers, small businesses, and contractors to manage their invoicing, customers, and product catalog completely offline.

Built with strict **Clean Architecture**, `flutter_bloc`, and `Isar` local database, this codebase is highly scalable, incredibly fast, and ready for you to reskin and publish to the App Store and Google Play.

## 🚀 Key Features
* **100% Offline-First:** Powered by Isar Database. No cloud subscriptions required.
* **Professional PDF Generation:** Create, preview, and share pixel-perfect A4 invoices natively.
* **Built-in CRM:** Manage a database of clients and customers for quick invoice generation.
* **Product & Service Catalog:** Save items, base prices, and default tax rates to speed up billing.
* **Dashboard & Analytics:** Real-time metrics for Total Revenue, Outstanding Balances, and Invoice lifecycle states.
* **Backup & Restore:** Export the entire database as a highly secure `.isar` file and restore it on any device.
* **Dynamic Currency System:** Supports localized currency formatting based on user preferences.
* **Modern Material 3 UI:** Premium fintech-grade design, fully responsive, with Dark Mode support.

## 🛠 Tech Stack
* **Framework:** Flutter 3.24+ (Dart 3+)
* **Architecture:** Feature-based Clean Architecture
* **State Management:** flutter_bloc (Cubits)
* **Local Storage:** Isar Database (v3)
* **Routing:** go_router
* **Dependency Injection:** GetIt
* **Functional Programming:** Dartz (Either/Failures)
* **PDF Engine:** pdf & printing packages
